Exemplos de aplicações para os microcontroladores PIC
========================
* Autor: César Magalhães
* Tecnologias: Proteus 8.4, mikroC PIC 6.2
* Resumo: Aplicações para os microcontroladores PIC
* Linguagens: Assembly, C (ANSI)
* Fonte: <https://github.com/cams7/exemplos-microcontroladores-pic>
* Linkedin: <https://br.linkedin.com/in/cams7>

Qual a finalidade desses exemplos?
-------------------
Esses exemplos foram estudados e testados com intuíto de aprender um pouco mais sobre microcontroladores, especialmente os PIC.

Sistemas requeridos
-------------------
* [Proteus 8.4] (https://www.labcenter.com/)
* [mikroC PIC 6.2] (http://www.mikroe.com/mikroc/pic/)

Para testa os exemplos
-------------------
* Instale o Proteus 8.4
* Instale o mikroC PIC 6.2
